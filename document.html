Adding Dependencies

To use PubChat, include following in the <body> tag on your page. 

<script src="https://cdn.pubnub.com/pubnub-3.5.1.min.js"></script>
<script src="http://code.jquery.com/jquery-1.8.2.min.js"></script>

Initializing PubNub

It is the next step to initialize the PubNub because PubChat works based on PubNub. 

function PubNub() {
    publishKey   : 'key',
    subscribeKey : 'key'
});

Connect user to PubChat

After initialize PubNub, you must connect user to PubChat with user's name. 

PubNub.connect = function(username) {
    this.username = username;
    this.connection = PUBNUB.init({
    	publish_key : this.publishKey,
    	subscribe_key : this.subscribeKey,
    	uuid : this.username
    });
};

Publish, Subscribe, History Functions

In PubNub, if you change anything, it is published to PubNub, and it is subscribed. When you cancel the change, it is unsubscribed. You can log all change with History Function.

For Example, now in your pubchat,
When you create new chat room, the room is published and subscribed with channel.
Now, you can use the room(really you use the channel).
When you chat with someone, you and he/she use the channel, you can get all chatting content with history function. 

PubNub.subscribe = function(options) {
    this.connection.subscribe.apply(this.connection, arguments);
    this.addSubscription(options.channel);
    this.saveSubscriptions();
};

PubNub.unsubscribe = function(options) {
    this.connection.unsubscribe.apply(this.connection, arguments);
};

PubNub.publish = function() {
    this.connection.publish.apply(this.connection, arguments);
};

PubNub.history = function() {
    this.connection.history.apply(this.connection, arguments);
};

In the above code, you can see addSubscription() and saveSubscriptions() functions. 

PubNub.addSubscription = function(channel) {
    this.subscriptions.push(channel);
    this.subscriptions = $.unique(this.subscriptions);
};

PubNub.removeSubscription = function(channel) {
    if (this.subscriptions.indexOf(channel) !== -1) {
        this.subscriptions.splice(this.subscriptions.indexOf(channel), 1);
    }
    this.saveSubscriptions();
};

PubNub.saveSubscriptions = function() {
    localStorage["pn-subscriptions"] = this.subscriptions;
};
